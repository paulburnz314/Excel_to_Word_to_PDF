# Excel_to_Word_to_PDF
Automate importing Excel data into a Word template and then create a PDF.  Make one document or make 100!  

Windows 10 Requirements include:

  Install the latest version of python, I am now using 3.8.

  For the Environmental Variable PATH the following is necessary (the python install may take care of this for you):
    C:\Users\paul\AppData\Local\Programs\Python\Python38-32\Scripts\
    C:\Users\paul\AppData\Local\Programs\Python\Python38-32\
  
  Install PyCharm Community 2020.1.2
  
  In the PyCharm terminal pip install the following (after activating the virtual environment):
    pip install comtypes
                wheel (not sure if this is absolutely necessary)
                openpyxl
                python-docx
                docx-mailmerge
